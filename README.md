# RLS-C6 v3

Готовый монолитный проект для ESP32-C6.

Что есть:
- сохранены API `/api/state`, `/api/set`, `/api/cmd`, `/api/save`, `/update`
- вкладки: Radar, Servo, Battery, Rules, Zones, Files, System
- профили, зоны, правила, редактор файлов, лог, OTA
- расширенные режимы сервопривода:
  - мягкое торможение у краёв
  - компенсация люфта
  - раздельная коррекция левой/правой стороны
  - автоцентрирование
  - табличная калибровка

Как собрать:
- положить `Esp32c6-radar.ino` в корень репозитория
- положить `.github/workflows/build.yml`
- GitHub Actions соберёт `.bin`

После первой прошивки:
- подключиться к `RLS-ESP32C6` / `12345678`
- открыть `http://192.168.4.1`
- редактировать UI и конфиги прямо через вкладку Files
- правила и зоны можно менять из вкладок Rules и Zones
- ядро прошивки обновляется через `http://192.168.4.1/update`
